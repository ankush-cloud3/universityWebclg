# 🎓 University Website Frontend

A modern, responsive frontend for a university website built using **React**, **Vite**, and **Tailwind CSS**.

---

## 🌐 Live Demo

🚀 **Preview the project here:**
👉(https://universitysitebms.netlify.app/)

> 🔁 Replace `#` with your deployed link (Vercel / Netlify / GitHub Pages)

---

## 🚀 Features

* ⚡ Fast and optimized with Vite
* 🎨 Modern UI using Tailwind CSS + Radix UI
* 📱 Fully responsive design
* 🧩 Reusable component-based architecture
* 🌙 Theme support (light/dark mode ready)
* 🔄 Client-side routing with React Router
* 📊 Interactive UI components

---

## 🛠️ Tech Stack

* **Frontend Framework:** React
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **UI Components:** Radix UI
* **Routing:** React Router

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone <your-repo-url>
cd university-website-frontend
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

👉 App runs at: `http://localhost:5173`

---

## 🏗️ Build for Production

```bash
npm run build
```
